# Diabetic Retinopathy Detection & AI Consultancy System

A comprehensive AI-powered system for detecting diabetic retinopathy using Machine Learning algorithms with an integrated chatbot for medical consultancy.

## Features

- **🖼️ Retinal Image Analysis**: Upload and analyze fundus images
- **🤖 Dual AI Models**: CNN for image classification + Random Forest for feature-based analysis
- **💬 AI Chatbot Consultancy**: Get answers about diabetic retinopathy
- **📊 Comparative Analysis**: Compare model performances
- **📈 Visual Analytics**: Interactive charts and reports

## Technologies Used

- **Python 3.12**
- **Streamlit** - Web application framework
- **TensorFlow/Keras** - CNN model implementation
- **Scikit-learn** - Random Forest classifier
- **OpenCV** - Image processing
- **Plotly** - Interactive visualizations
- **Pandas/Numpy** - Data manipulation
